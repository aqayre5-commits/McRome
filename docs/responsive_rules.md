# Responsive Rules

| Rank | Rule | Mobile | Tablet | Desktop |
|---|---|---|---|---|
| 1 | Header density | Logo + 2 actions max | Full nav | Full nav |
| 2 | Hero layout | Single column | Single column | Two-column split |
| 3 | Directory grid | 1 column | 2 columns | 3 columns |
| 4 | Game detail layout | Content then CTA | Content then CTA | 2-column with sticky sidebar optional |
| 5 | Forms | Full width stacked | Full width stacked | Inline when one primary input |
| 6 | Touch targets | 44px min height | 44px min height | 40px min height |
| 7 | Card padding | 20px | 24px | 24px |
| 8 | Text line length | <= 34 chars avg | <= 42 chars avg | <= 70 chars max |
| 9 | Tables in docs/admin | horizontal scroll | horizontal scroll | full width |
| 10 | FAQ interactions | collapsed by default | collapsed by default | collapsed by default |
