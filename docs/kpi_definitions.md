# KPI Definitions

| Rank | KPI | Formula | Target direction |
|---|---|---|---|
| 1 | Published page count | count of `roblox_pages.is_published=true` | up |
| 2 | Indexed page ratio | indexed pages / published pages | up |
| 3 | Organic landing-to-detail CTR | detail page sessions from organic / landing organic sessions | up |
| 4 | Search usage rate | `search_submitted` / directory sessions | up |
| 5 | Save rate | `game_saved(intent=save)` / detail page sessions | up |
| 6 | Login completion rate | dashboard sessions / `login_requested` | up |
| 7 | Checkout conversion rate | `checkout_completed` / `pricing_viewed` | up |
| 8 | 7-day returning saved-user rate | returning paid users / paid users | up |
